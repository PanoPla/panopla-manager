Easily create panos on your WordPress website, add virtual tours and more.

== Description ==

PanoPla is a CMS for creating and managing your virtual tours, virtual and augmented reality projects, and annotating huge images. This plugin helps you add that content to your WordPress blog.

== Installation ==

1. Install the plugin
2. Get the ID of the pano you want to embed
3. Create the shortcode on your site
4. Load page

== Frequently Asked Questions ==

= Do I need a PanoPla account? =

No! You can embed a pano on your site easily even without an account.
